# 牧原股份 (002714) — 猪周期分析

## 用户核心判断（本轮猪周期定性，需持续验证）

**假设：本轮周期呈 W 型双底，而非传统 V 型反转。**

1. **底部格外长、格外深** — 规模化养殖占比提升，大企业扛亏能力强，去化极其缓慢
2. **投资者对猪周期格外了解** — 价格接近前低时就会有人抢跑抄底，制造短期低点和反弹
3. **W 型第一底（当前）** — 猪价接近历史极端低位，抄底资金进场后会有反弹
4. **W 型第二底（H2 2026）** — 原油上涨引发饲料通胀（玉米/豆粕涨价），养殖成本上升，亏损放大，倒逼去化加速
5. **真正的行情在第二底之后** — 去化完成后，猪价有翻倍空间

## 行业知识

**猪周期逻辑**：约 4 年一轮（上行 18-24 月，下行 18-24 月）。
```
猪价涨 → 利润高 → 补栏扩产 → 供给过剩 → 猪价跌
   ↑                                            ↓
   └── 能繁母猪去化 ← 深度亏损 ← 供给过剩 ←──────┘
```

**核心观测指标**：

| 指标 | 含义 | 领先/同步 | 关键阈值 |
|------|------|----------|---------|
| 能繁母猪存栏同比 | 领先出栏 10 个月 | 领先 | 同比 -5% 以上 = 去化加速 |
| 猪粮比（生猪/玉米） | 养殖盈亏线 | 同步 | <6:1 = 全行业亏损 |
| 自繁自养头均利润 | 行业盈利状况 | 同步 | < -200 元 = 深度亏损 |
| 仔猪价格 | 补栏意愿 | 领先 5-6 月 | 暴跌 = 散户退出 |
| 玉米/豆粕价格 | 饲料成本（本轮第二底催化剂） | 领先 | 持续上涨 = 成本推动去化 |

**关键拐点信号**：
- 能繁母猪存栏持续下降（去化加速）
- 猪粮比跌破 6:1（全行业亏损）
- 仔猪价格暴跌（养殖户停止补栏）
- 养殖企业开始淘汰母猪

**风险点**：
- 规模化养殖占比提升拉长底部时间（大企业更扛亏）
- 牧原高杠杆扩张，猪价低迷时财务压力大
- 能繁母猪没开始去化就不是底
- 第一底反弹后不要追——第二底才是真正机会

## 牧原技术指标的独特解读

猪周期底部的牧原，技术指标含义与普通股票不同：

### RSI：极端超卖是底部常态，不是买入信号

- 猪价跌 → 股价跌 → RSI 跌，三者同步。RSI < 20 在猪周期底部**反复出现**（历史共 77 天，<15 共 29 天）
- RSI 连续 5+ 日 <20 是**恐慌性抛售（capitulation）**标志，接近底部但未必是最低点
- **在猪周期里，不能等底背离再动手**——等到底背离出现，猪价可能已涨 20%，股价已反弹 30%+
- 正确的读法：RSI 从 <15 回升到 25+ 且不再回落到 <20，才是企稳信号

### 成交量：周期底部放量 = 机构左侧建仓

- 普通个股底部是"缩量筑底"，**牧原的底部经常放量**
- 因为机构投资者知道猪周期会反转，会在底部区域持续吸筹
- "下跌中正常量"或"放量下跌"在牧原身上≠坏事，可能是聪明钱在接
- 真正危险的是**连续缩量阴跌**——说明连抄底资金都不来了

### 换手率：最好的"底部情绪"指标

- 换手率从极度低迷（<0.8%）跳升到正常（>1.2%）+ 价格止跌 = 有人开始接货
- 如果换手率持续 > 1.5% + 价格横盘 = 底部筹码交换活跃，接近反转
- 换手率比 RSI 更可靠，因为它反映的是"谁在买"而非"超卖了多少"

### 均线：空头排列是猪周期底部的标配

- 猪周期底部，空头排列是正常状态，不值得恐慌
- 关键不是"均线是否多头"，而是"价格是否开始收敛到 MA20 附近"
- 价格从偏离 MA20 -17% 收窄到 -5% 以内 → 筑底进行中

### 总结：牧原 vs 普通股票的技术指标权重

| 指标 | 普通股票权重 | 牧原底部权重 | 原因 |
|------|------------|------------|------|
| RSI | 高 | 中低 | 猪周期底部 RSI 长期低迷，信号噪音多 |
| 成交量 | 中 | 中高 | 机构左侧建仓会放量，不是坏事 |
| 换手率 | 低 | **高** | 反映底部筹码交换，最干净的信号 |
| MACD 死叉收敛 | 中 | 中高 | 猪周期底部 MACD 收敛 = 杀跌动能衰竭 |
| 均线 | 高 | 中 | 空头排列是猪周期底部常态 |
| 底背离 | **高** | 低 | 出现太晚，等它会错过最佳买点 |

| 阶段 | 操作 | 仓位 |

- 去化未完成，反弹是交易性机会而非趋势反转
- 第二底可能比第一底更低（饲料通胀 + 加速去化）
- 第一底买入被套，心态会被动，错过第二底的真正机会

**极端情况预案**：
- 如果 PB 分位跌到 5% 以下 + RSI < 10 + 猪粮比 < 5:1 → 赔率极度倾斜，可考虑在第一底加至 1/3 仓
- 如果第一底反弹超过 30% → 警惕 V 型反转可能，不要机械等待第二底
